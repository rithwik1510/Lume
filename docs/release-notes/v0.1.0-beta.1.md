# Lume v0.1.0-beta.1

First public beta. **Windows only.**

Lume hosts a fleet of AI coding agents in parallel across smooth tiled terminal
panes, with a markdown editor and a localhost preview — one workstation for
agent-driven development.

## Install

- **One-liner:** `npx lume-desktop`
- **Direct download:** grab `Lume_0.1.0-beta.1_x64-setup.exe` below and run it.

> The installer is **unsigned** during the beta, so Windows SmartScreen shows
> "Windows protected your PC." Click **More info → Run anyway**. Installs per-user;
> no admin needed.

## Highlights

- Smooth tiled terminal panes (xterm.js + WebGL) on real PTYs — 32 ms batched
  IPC, 8 MB per-pane ring buffer.
- Session manager sidebar — grouped sessions, rename, attention glow when a
  background agent goes quiet.
- Session restore — reopens your last layout and pre-fills each pane's remembered
  first command (never auto-run).
- Markdown editor (CodeMirror 6) + MD Quick Viewer.
- Localhost Preview panel — iframe a dev server beside your terminals.
- Drag a file from Explorer or the file drawer onto a terminal.
- Settings UI with theme + font-pair presets; hot-reloaded `config.toml`.
- In-app auto-update.

## Known limitations

- Windows only this beta; macOS/Linux later.
- Installer is unsigned (SmartScreen warning — see above).
- PTYs do not survive restart; sessions revive layout + pre-filled commands, not
  live processes.
