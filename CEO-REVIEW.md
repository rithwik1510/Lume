## Vision

### 10x Check
The 10x version is an **agent operating system for solo developers**: a workstation where the dashboard isn't just "list of panes" but a true fleet view that shows each agent's current task, files it has touched in this session, exit codes, idle vs thinking detection, and the ability to route a task from one pane to another or fork an agent into two parallel branches. Markdown notes become a first-class plan/log surface that reference live panes. Cross-machine state sync turns "my workstation" into a portable identity. This is 3-5 quarters of work, not weeks. The v0.1 plan is a deliberate wedge into this larger product.

## Premise Challenge Resolved

The CEO review challenged four premises:

1. **Is the terminal the right primitive?** Defended by user: Claude Code and Codex are TUIs, so terminal IS the right primitive. Locked.
2. **Is "desktop app" the right form factor?** Defended by user: full workflow shift requires owning the chrome; extensions can't deliver the "desk" metaphor. Locked.
3. **Should the MVP be inverted (dashboard first)?** User chose to defer dashboard to v0.1.1 — ship foundation first. Locked.
4. **Where's the second user?** User chose to gate marketing surface (landing/GIF) on "does the product feel good first." Locked.

All four premises were defended with concrete reasoning. The framing is the user's, not borrowed from the review.
